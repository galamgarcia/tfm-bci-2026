# Desarrollo de un sistema de interacción BCI aplicado a videojuegos
Aplicación móvil desarrollada en Unity compatible con BrainLink Pro para la interacción mediante Interfaces Cerebro-Computadora (BCI) en videojuegos. Procesa señales EEG no invasivas del lóbulo frontal para identificar patrones cognitivos y convertirlos en acciones dentro del juego.
